Cunny Learning Materials API

Get All Learning Materials

This API endpoint retrieves all learning materials from the database.

Request

Method: GET

URL: /learning-materials

Response

                        
                            {
                                "error" : false,
                                "message" : "Learning material retrieved successfully.",
                                "learningMaterial" : {
                                    "id" : 1,
                                    "title" : "Introduction to Node.js",
                                    "name" : "Node.js Basics",
                                    "learning_image_path" : "https://storage.googleapis.com/my-bucket-name/images/nodejs-intro.jpg",
                                    "body" : "This is the body of the learning material.",
                                    "created_at" : "2024-01-01T00:00:00Z",
                                    "updated_at" : "2024-01-01T00:00:00Z"
                                }
                            }

Get Learning Material by ID

This API endpoint retrieves a learning material by its unique ID.

Request

Method : GET

URL : /learning-materials/{id}

# Replace {id} with the ID of the learning material you want to retrieve.

Response

                        
                            {
                                "error" : false,
                                "message" : "Learning material retrieved successfully.",
                                "learningMaterial" : {
                                    "id" : 1,
                                    "title" : "Introduction to Node.js",
                                    "name" : "Node.js Basics",
                                    "learning_image_path" : "https://storage.googleapis.com/my-bucket-name/images/nodejs-intro.jpg",
                                    "body" : "This is the body of the learning material.",
                                    "created_at" : "2024-01-01T00:00:00Z",
                                    "updated_at" : "2024-01-01T00:00:00Z"
                                }
                            }

Bad Request & Response

                        
                            {
                                "error" : true,
                                "message" : "Learning material not found. No material found with ID: {id}"
                            }
